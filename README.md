## 📖 关于本项目

这是一个用纯 HTML5 + SVG 绘制的搞笑四格漫画网页，讲述了两只狗狗——**旺财**和**小白**——的日常趣事。无需任何框架，打开浏览器即可欣赏。

---

## 🐶 角色介绍

| 角色 | 毛色 | 性格特点 |
|------|------|---------|
| **旺财** | 棕色，耳朵竖立、尾巴翘起 | 憨厚但健忘，爱挖坑埋骨头 |
| **小白** | 白色，耳朵耷拉、脸带腮红 | 活泼幽默，喜欢自嘲，与旺财形成反差萌 |

---

## 📚 故事梗概

**第 1 格 · 旺财的困惑**
> "我今天挖了个坑，埋了块骨头，然后……忘了埋哪！"

绿草地上，旺财奋力挖坑，泥土四溅，头顶大大的问号。

**第 2 格 · 小白的共鸣**
> "哈哈！我也是！我埋了三十七块骨头，只记得那个数字 😂"

小白笑到眼睛弯成弧线，尾巴疯狂摇晃。

**第 3 格 · 主人的误会**
> "主人说我们是'聪明的狗'……他不知道我们连自己零食都找不回来 😅"

远处主人满脸骄傲，旺财流汗大眼望天。

**第 4 格 · 默契的密谋**
> "嘘——别说！只要尾巴摇得够快，他们什么都信！🐾"

两只狗狗并肩挤眼，中间浮现爱心，阴谋得逞的表情一模一样。

---

## ✨ 技术亮点

- **纯 HTML5 + SVG**：零依赖，无需 Node.js / 构建工具，下载即用
- **手绘漫画风格**：圆角格子、粗边框、阴影，还原真实四格漫画质感
- **SVG 动态细节**：尾巴摇晃用多重描边模拟，泥土飞溅用散点圆形表现
- **响应式布局**：`flex` + `grid` 双布局，手机和桌面都好看
- **对话气泡**：椭圆+三角尖角设计，指向每只狗狗的嘴巴
